# Support

ForgeMark is an open-source marketing automation platform maintained by Buildly Labs.

## Support Scope & SLA

### Community Support (Free)

- **Response Time Goal:** 48 hours for community channels
- **Scope:** Installation, configuration, basic troubleshooting, and feature requests
- **Duration:** Best-effort support for all community users

### Labs Premium Support (Optional)

Users can access premium support through the Buildly Forge marketplace:

- **Response Time Goal:** 24 business hours
- **Scope:** Priority bug fixes, deployment assistance, custom integration guidance
- **Duration:** 30 days from purchase date

## Support Channels

### GitHub Issues (Community)
- **Primary support channel**
- Report bugs, request features, ask questions
- GitHub repository: https://github.com/buildlyio/marketing
- **Expected response:** 24-48 hours

### Discord (Community)
- **Secondary support channel**
- Real-time community discussions
- Join: https://discord.gg/buildly
- **Best for:** quick questions, troubleshooting tips from community

### Email (Labs Premium Only)
- **For premium support subscribers**
- Email: support@buildly.io
- **Subject line:** Include "ForgeMark" and brief issue title
- **Response:** Within 24 business hours

## What's Covered

✅ **Included:**
- Installation and setup guidance
- Configuration assistance
- Basic troubleshooting
- Feature documentation
- Community bug reports and fixes

❌ **Not Included:**
- Custom feature development
- Custom integrations (unless covered by premium support)
- Performance optimization consulting
- Third-party tool integration support (Brevo, Google Ads, etc.)
- On-site or dedicated support

## Reporting Issues

### For Bug Reports
1. Check existing GitHub issues first
2. Include:
   - ForgeMark version and Python version
   - Operating system and environment
   - Steps to reproduce
   - Error messages or logs
   - Expected vs. actual behavior

### For Feature Requests
1. Open a GitHub discussion or issue
2. Describe the use case and desired outcome
3. Provide context on how it benefits multi-brand marketing

## Security Issues

**Do not report security vulnerabilities publicly.** Email security@buildly.io with:
- Vulnerability description
- Affected version(s)
- Steps to reproduce
- Potential impact

Security vulnerabilities will be addressed within 30 days and credited to reporters.

## Version Support

- **Latest version:** Full support for current release
- **Previous versions:** 6-month bugfix support window
- **Older versions:** Use of latest version is strongly recommended
