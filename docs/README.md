## Quick Reference

### Brand Configuration
- Multi-brand system supporting: Buildly, Foundry, Open Build, Radical Therapy, Oregon Software
- Each brand has unique branding, colors, fonts, and messaging

### Key Components
1. **Dashboard System** - Flask-based web interface (`/dashboard/`)
2. **Email Templates** - Brand-specific HTML email templates (`email_templates.py`)
3. **Analytics System** - Real-time analytics and reporting (`/automation/`)
4. **Outreach Automation** - Multi-brand outreach campaigns (`/automation/`)
5. **Social Media** - Automated social media posting (planned)
