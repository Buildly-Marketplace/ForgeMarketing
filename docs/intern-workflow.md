# Intern Workflow

This guide is for day-to-day marketing operations in ForgeMarketing.

## Daily Workflow

1. Open the dashboard and check Due Today.
2. Review items in Needs Review and Ready to Schedule.
3. Complete manual tasks due today (posting, approvals, asset prep, platform updates).
4. Add posted URLs immediately after publishing.
5. Add short notes for blockers and handoff status.

## Weekly Workflow

1. Confirm active campaigns and priorities.
2. Plan 1 week of content ideas by pillar.
3. Convert ideas into platform-specific drafts.
4. Run a review and approval pass.
5. Schedule approved posts in your external scheduler.
6. Complete analytics checks for 24h and 7d snapshots.
7. Prepare weekly summary and next actions.

## Create a Campaign

1. Select brand/product.
2. Add campaign name, objective, date range, and CTA.
3. Select target platforms.
4. Set owner and notes.
5. Save campaign in planning status.

## Draft Content

1. Add a content idea with title, message, audience, and CTA.
2. Assign owner and due date.
3. Add source notes and required assets.
4. Set status to idea or drafted.

## Create Platform-Specific Posts

For each content idea, create one draft per platform.

Suggested outputs:
- Instagram caption
- TikTok script
- YouTube Shorts title and description
- X/Bluesky short version
- LinkedIn long-form version

## Manual Tasks

Use manual tasks for anything requiring human action:
- account setup
- brand/profile updates
- approvals
- manual posting
- external scheduling
- analytics checks

Each task should include:
- owner
- due date
- checklist steps
- status
- notes and links

## Scheduling with External Tools

ForgeMarketing supports scheduler handoff workflows.

Recommended process:
1. Move draft to approved.
2. Add scheduler details (Buffer, Metricool, Later, Hootsuite, or manual posting note).
3. Schedule externally.
4. Update item to scheduled.
5. After publish, paste posted URL and mark posted.

## Record Posted URLs

After publishing, add:
- live post URL
- posted timestamp
- final caption/version used
- notes (edits made on platform)

## Analytics Checks

Run checks at:
- 24 hours
- 7 days
- optional 30 days

Track:
- impressions/views
- likes
- comments
- shares/reposts
- saves
- clicks
- follows/subscribers

## Weekly Report Checklist

1. Content created this week
2. Content posted this week
3. Overdue items
4. Missing assets
5. Best-performing posts
6. Next week priorities
7. Risks/blockers
