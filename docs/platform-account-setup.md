# Platform Account Setup (Manual Checklist)

Important: These are manual checklist templates. Do not automate account creation or verification.

## TikTok

- Create account manually with approved brand email.
- Add profile image.
- Add bio.
- Add website link if available.
- Enable 2FA.
- Store account ownership notes securely outside this app.
- Add account URL to brand profile.
- Mark setup task complete.

## YouTube

- Create or connect brand channel manually.
- Add profile image and banner.
- Add product/company description.
- Add website and social links.
- Verify channel if needed.
- Add manager/intern permissions if applicable.
- Add channel URL to brand profile.
- Mark setup task complete.

## Instagram

- Create account manually or via Meta Business Suite.
- Add profile image, bio, and website link.
- Convert to professional account if needed.
- Connect Meta Business Suite if used.
- Enable 2FA.
- Add account URL to brand profile.
- Mark setup task complete.

## X

- Create account manually.
- Add profile image, banner, bio, and URL.
- Enable 2FA.
- Add account URL to brand profile.
- Mark setup task complete.

## Bluesky

- Create account manually.
- Add profile image, bio, and URL.
- Optional later: move to custom domain handle.
- Add account URL to brand profile.
- Mark setup task complete.

## Threads

- Create/connect through Instagram manually.
- Add profile details.
- Add account URL to brand profile.
- Mark setup task complete.

## LinkedIn

- Create company/product page manually.
- Add logo, banner, overview, and website.
- Add admins.
- Add page URL to brand profile.
- Mark setup task complete.

## Pinterest

- Create account manually.
- Add profile image, bio, and website.
- Verify website when ready.
- Add account URL to brand profile.
- Mark setup task complete.

## Safety Note

ForgeMarketing helps teams plan and manage content. Account creation, verification, and final publishing should be done by authorized humans following platform terms.
