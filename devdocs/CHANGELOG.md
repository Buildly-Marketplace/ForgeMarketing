# Changelog

All notable changes to this project are documented here.

## [1.0.0] - 2025-11-14

### Added
- Initial stable release (v1.0.0)
- Multi-brand marketing dashboard
- AI content generation via Ollama
- Email marketing automation with Brevo integration
- Social media publishing (Twitter, Mastodon, Bluesky)
- Google Ads campaign management
- Real-time analytics dashboard
- Influencer discovery engine
- Contact management system
- Campaign scheduling and automation
- Docker and Kubernetes deployment support
- GitHub Actions CI/CD pipeline
- Comprehensive API documentation
- Smoke tests for health checks and CRUD operations

### Changed
- Project restructured to Buildly Way standards
- Documentation moved to centralized `devdocs/` directory
- Licensing changed to BSL-1.1 → Apache-2.0 (change date: 2026-11-14)

### Deprecated
- None

### Removed
- None

### Fixed
- Email delivery webhook tracking
- Multi-brand contact deduplication logic
- Timezone handling in campaign scheduling
- Ollama connection retry logic

### Security
- Secret key generation for Flask sessions
- API key validation for third-party integrations
- Database connection encryption for PostgreSQL
